# CLAUDE.md — hoehenvergleich-app

## Sprache & Schreibweise (Birchmeier-Standard, verbindlich)
- **Deutsch, Schweizer Rechtschreibung.** Alle UI-Texte, Beschriftungen, Kommentare und Doku auf Deutsch.
- **Echte Umlaute** ü/ä/ö und Ü/Ä/Ö verwenden — NIE die Transliteration ue/ae/oe.
- **Kein ß (Eszett):** Schweizer „ss" bleibt (z. B. „Strasse", „muss", „gross").
- **Immer UTF-8** als Datei-Encoding, damit Umlaute korrekt gespeichert werden.

## Weitere Birchmeier-Standards
- Keine Emojis in UI, Code oder Doku — stattdessen Text-Labels oder einfache Unicode-Zeichen.
- Keine nativen Browser-Defaults (kein alert/confirm/ungestyltes select) — Custom-UI passend zum Layout.
- Kompakte, mehrspaltige Layouts; wenig Leerraum und Scrollen.
